# Heart Rate LEDs
LED lighting controlled by your Heart Rate using a Pimoroni Plasma 2350 W

This project visualises your heart rate via WS812/NeoPixel LED lighting. It runs on a Pimoroni Plasma 2350 W board, which is searching for a locally broadcast Bluetooth heart rate.

Adjust the code to take account of the number of LEDs you are using and your Resting Heart Rate and Maximum Heart Rate.

Full details at https://www.adambowie.com/blog/2026/03/led-lighting-controlled-by-your-heart-rate/
